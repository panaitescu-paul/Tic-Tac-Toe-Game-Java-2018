# Tic Tac Toe Game - Java - 2018
### Tic Tac Toe Game that runs in a Browser with AI decisions.

## Features: 
- **Login/Logout**
- **Game Logic**
	- Register Wins, Losses, Ties
	- Detect winning conditions
	- **Computer player - AI decisions**
	- Reset board
- **Leaderboard**
- **Database implementation**

## Technologies:
- Java
- Spring framework
- HTML
- CSS

## Tools:
- IntelliJ IDEA
